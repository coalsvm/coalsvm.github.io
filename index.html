CoalSVModel v3.1
A MATLAB GUI for Coal Seam Modelling with SVM

This site is about CoalSVM v3.1, its documentation, and How-to guides.

The main objective of this standalone executable program is to generate lignite block model with support vector machines (SVM) methodology. The outputs of the application are thickness map, top contours of the lignite seam, and block centroids which can be exported in various formats, such as Fixed-format ASCII, Geo-EAS ASCII, and GEOVIA Surpac. CoalSVModel is coded with MATLAB® programming and numeric computing platform and made executable with MATLAB® Compiler™ application.
